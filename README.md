# Roadtrip 2026 — Winnipeg to Toronto

Father-son roadtrip, June 29 – July 2, 2026.

Single-page itinerary site: route map (Leaflet + Wikipedia tiles), 4-day plan, and trip budget.

## Live site

https://fadibahodi.github.io/roadtrip-2026/

## Run locally

Just open `index.html` in a browser. No build step, no API keys, no dependencies beyond the CDN-hosted Leaflet library.

## Stack

- Plain HTML/CSS
- [Leaflet](https://leafletjs.com/) for the map
- Wikipedia map tiles (no API key required)
